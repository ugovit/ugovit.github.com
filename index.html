Aswatna

Aswatna أصواتنا is a smart application developed by ugovit* to create a two-way medium for information and knowledge.
The idea was developed within the concept of electronic democracy, in which users can easily voice their opinions on various matters after getting insight on these issues through reading articles and following up with news. The Aswatna project (Our Voices) started in December 2013. Since then, the application has went through a vigorous process of development and enhancement.

We hope that this app become a small, but essential brick, in the building and strengthening of democracies.

Monthly Questions

Designed to give administrator a monthly update on their performance based on the users answers. There is an attachment feature available that enables the administrator to attach a list of accomplishments or information for their users along with the question.

Weekly Questions

Tailored for rating issues or polling the users on any subject. This feature can be a tool to test ideas and build a discussion.

Comments

Allows the user to comment and offer suggestions. Simply, it provides a space where the user can write what they want to the administrator while reserving the right to remain anonymous.

Articles

This feature allows the administrator to send notes, articles, announcements, alerts, etc., of up to 400 words preferably). It also allows the user to like/dislike the text sent. This gives prompt feedback to the administrator.

Archives

Aswatna provides users with a color coded archive where they will find all previous questions and their results, which can help users track responses and identify trends.

Upload Picture

Aswatna provides users with the ability to send pictures and sustain users’ interaction with the application.
